## 1. Executive Summary & Vision
"OneTap Admin" is a headless, serverless "Zero-UI" WhatsApp personal administrative assistant designed for ordinary people (parents, students, renters, freelancers) to bypass the "Life Admin Tax." 
Instead of clunky apps or manual print-fill-scan cycles, users forward any flat/non-interactive PDF, form, or document link to a WhatsApp business number. The backend automates document parsing, maps standard user profiles, dynamically queries the user for missing fields via interactive WhatsApp messages, stamps signatures, and returns a fully compiled PDF in seconds.

---

## 2. Technical Stack
- **Gateway:** Twilio / Meta WhatsApp Business API (`POST /api/webhook/whatsapp`).
- **Database:** Supabase (Postgres) for secure user profile data and session state tracking.
- **Cognitive Engines:** 
  - OpenAI GPT-4o (Vision for layout parsing and dynamic Q&A generation).
  - Web Agent (Future expansion: stagehand / browser-use / Browserbase for automating web form links).
- **PDF Generation:** PDF.co API (manipulating flat PDF coordinates, injecting annotations, and stamping transparent PNG signatures).
- **Deployment Platform:** Render (Serverless Node.js/TypeScript Express backend).
